### 6. `server.js` — Adicionar rota para listar vendedores

Nova rota `GET /api/vendedores` que retorna lista de usuários com `cargo === "vendedor"`:
```js
app.get('/api/vendedores', requireAuth, (req, res) => {
  const vendedores = Object.entries(usuarios)
    .filter(([_, u]) => u.cargo === 'vendedor')
    .map(([login, u]) => ({ login, nome: u.nome }));
  res.json(vendedores);
});
```

### 7. `public/js/funcionario.js` — CRM: vendedor como select

**Alteração na aba CRM:**
- Substituir `<input type="text" id="crm-vendedor" ...>` por `<select id="crm-vendedor">`
- Na inicialização da aba CRM, buscar `/api/vendedores` e popular o select:
  ```js
  async function carregarVendedores() {
    const resp = await fetch('/api/vendedores', { headers: { 'Authorization': 'Bearer ' + token } });
    const vendedores = await resp.json();
    const select = document.getElementById('crm-vendedor');
    select.innerHTML = '<option value="">Selecione o vendedor</option>';
    vendedores.forEach(v => {
      select.innerHTML += `<option value="${v.nome}">${v.nome}</option>`;
    });
  }
  ```
