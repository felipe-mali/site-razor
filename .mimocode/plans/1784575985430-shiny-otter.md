# Razor Indústria — Public Site Visual Refinement Plan

## Context
The public site already has: GSAP ScrollTrigger + Lenis, Three.js hero particles, Motion.dev microinteractions, SVG icons, scoped premium.css. Now we refine the visual details — typography, spacing, hero impact, card polish, footer, and mobile experience.

## Constraints
- No changes to: backend, APIs, auth, business logic, JS functionality
- Only CSS and HTML changes
- All libraries already loaded (GSAP, Lenis, Three.js, Motion.dev)

## Phase 1: Hero Section Impact

### 1.1 Typography refinement
**File: `public/css/premium.css`** — Hero section

Current: h1 uses `clamp(3rem, 8vw, 6rem)` which is good. The "corta" accent has a gradient + underline.

Improvements:
- Increase hero h1 weight to 900 for more impact
- Add subtle text-shadow on h1 for depth
- Make the underline accent on "corta" animate in with GSAP
- Increase hero-desc max-width for better readability
- Add subtle letter-spacing to hero-badge

```css
.public-site .hero h1 {
  font-weight: 900;
  letter-spacing: -0.02em;
}

.public-site .hero-desc {
  max-width: 520px;
  font-size: 16px;
}

.public-site .hero-badge {
  letter-spacing: 0.15em;
  font-size: 11px;
}
```
