# Plano de Redesign - Razor Indústria

## Objetivo
Reformatar o site para combinar com o estilo do exemplo enviado:
- Visual escuro, angular, profissional
- Cores: #191919 (base) + #FDA61A (accent)
- Formulário avançado de orçamento com seletor de produtos
- Botão flutuante do WhatsApp
- Google Maps integrado

## Arquivos a Modificar

### 1. `public/css/style.css`
**Mudanças:**
- Trocar fonte para Barlow Condensed (display) + Barlow (body)
- Fundo escuro em todas as seções (#0A0A0A, #111214, #1A1C1F)
- Border-radius zero ou mínimo (2-4px máximo)
- Adicionar grid pattern no hero
- Estilo dos cards de produto com hover reveal
- Formulário com seletor de produtos, checkboxes, range sliders
- Botão WhatsApp flutuante
- Estilo do mapa

### 2. `public/index.html`
**Mudanças na estrutura:**
- Hero: adicionar grid pattern animado
- Produtos: cards com hover reveal (descrição aparece no hover)
- Seção Cotação: formulário avançado com seletor de produtos
- Contato: cards de contato + Google Maps
- Footer: adicionar logo e estrutura do exemplo
- Adicionar WhatsApp float button

### 3. `public/js/app.js`
**Novas funcionalidades:**
- Construção dinâmica do seletor de produtos
- Toggle de produtos com campos específicos
- Geração de mensagem formatada para WhatsApp
- Preview da mensagem antes de enviar
- Envio para WhatsApp com todos os dados

## Estrutura do Formulário Avançado

### Produtos com campos específicos:
1. **Concertina**: Tipo, Diâmetro, Metragem (range), Instalação sobre
2. **Tela Alambrado**: Altura, Malha, Bitola, Acabamento, Metragem, Postes
3. **Rede Laminada**: Altura, Metragem
4. **Tela Soldada**: Bitola, Malha, Altura, Metragem
5. **Gradil**: Altura, Módulo, Metragem, Postes
6. **Lança Protetora**: Modelo, Material, Comprimento, Espaçamento, Metragem
7. **Tela Artística**: Material, Aplicação, Medidas, Quantidade
8. **Outro**: Descrição livre

### Seções do formulário:
1. Dados pessoais (Nome, Empresa, Cidade/Estado)
2. Seleção de produtos (checkboxes expandíveis)
3. Informações gerais (Aplicação, Prazo, Observações)
4. Preview da mensagem
5. Botão enviar WhatsApp
