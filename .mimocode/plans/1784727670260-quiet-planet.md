# Plano: Modificar Módulo de Cotações para Estilo Planilha Excel

## Objetivo
Modificar a página de cotação de preços (Logística) para ficar com o funcionamento visual e estrutural conforme a imagem de referência - uma planilha de cotação estilo Excel clássico.

## Estilo Híbrido
- **Editor web**: Manter estilo escuro atual (melhor para uso prolongado)
- **Documento de impressão**: Estilo claro branco como na imagem (melhor para impressão)

## Definições Importantes
- **Desconto Sugerido**: Calculado automaticamente (diferença entre total recomendado e custo ideal)
- **Informações dos Fornecedores**: Apenas o NOME na tabela (Razão Social, CNPJ, etc. ficam nos cards)

## Análise da Imagem vs Implementação Atual

### Imagem mostra:
- Cabeçalho com NOMES dos fornecedores (A, B, C)
- Tabela principal com colunas: ITEM, QUANTIDADE, VALOR UNIT. e VALOR TOTAL para cada fornecedor
- Coluna CUSTO IDEAL com fundo verde claro
- Linha de DESCONTO SUGERIDO com fundo laranja (calculado automaticamente)
- Totais por fornecedor na parte inferior
- Seção de DESCRIÇÃO/RESUMO DA COMPRA
- Locais para assinaturas (ELABORADO POR, APROVADO POR)

### Implementação atual:
- Layout moderno escuro com cards separados
- Tabela com estilos diferentes
- Resumo financeiro em cards
