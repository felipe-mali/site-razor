# Plano: CRM Dropdown Vazio + Fuso Horário (UTC -> BRT)

## Problemas

### 1. CRM: dropdown de vendedor vazio ao editar cliente
- **Causa raiz**: A função `carregarVendedoresSelect()` (funcionario.js:1375) faz fetch para `/api/vendedores` com o token. Se a chamada falhar (token expirado, sessão perdida, etc.), o catch silencioso (`console.warn`) deixa o select só com "Selecione o vendedor" sem opções.
- **Possível causa adicional**: O select mostra `v.login` (ex: "miqueias") em vez de `v.nome` (ex: "Comercial 1"). Mesmo que funcione, o usuário pode não reconhecer.
- **Solução**: 
  1. Mostrar `v.nome` como texto da opção (ex: "Comercial 1") e `v.login` como value
  2. Adicionar fallback: se a API falhar, popular o select com os nomes dos vendedores dos clientes já cadastrados no CRM (IndexedDB)
  3. Adicionar log de erro visível no console para debug

### 2. Fuso horário: datas mostradas com +3h (UTC em vez de BRT)
- **Causa raiz**: `new Date().toISOString()` retorna UTC (GMT+0). No Brasil (GMT-3), são 23:55 e o ISO mostra 02:55 do dia seguinte.
- **Locais afetados** (todos usam `new Date().toISOString()`):
  - `funcionario.js` linhas 448, 849, 974, 1418, 1422, 1448, 1487, 1488
  - `cancelamentos.js` linha 219
- **Solução**: Criar função `agora()` que retorna Date ajustado para BRT (UTC-3), e substituir todas as chamadas `new Date().toISOString()` por `agora().toISOString()`. Também ajustar `formatarData()` e `formatarDataInput()` em cancelamentos.js.

```javascript
// Fuso horário Brasil (UTC-3)
function agora() {
  const d = new Date();
  d.setMinutes(d.getMinutes() - d.getTimezoneOffset() - 180);
  return d;
}
```
**Nota**: Usar `d.setMinutes(...)` para ajustar sem dependência de Intl. Alternativa mais robusta: `new Date(Date.now() - 3*60*60*1000)`.

#### C) Corrigir `carregarVendedoresSelect` (linha ~1375)

```javascript
async function carregarVendedoresSelect(vendedorAtual) {
  const select = document.getElementById('crm-vendedor');
  if (!select) return;

  try {
    const resp = await fetch('/api/vendedores', {
      headers: { 'Authorization': 'Bearer ' + token }
    });
    if (!resp.ok) throw new Error('HTTP ' + resp.status);
    const vendedores = await resp.json();

    select.innerHTML = '<option value="">Selecione o vendedor</option>';
    vendedores.forEach(v => {
      const selected = v.login === vendedorAtual ? 'selected' : '';
      select.innerHTML += `<option value="${v.login}" ${selected}>${v.nome || v.login}</option>`;
    });
    if (vendedorAtual && !vendedores.some(v => v.login === vendedorAtual)) {
      select.innerHTML += `<option value="${vendedorAtual}" selected>${vendedorAtual}</option>`;
    }
  } catch (err) {
    console.warn('Erro ao carregar vendedores da API:', err);
    // Fallback: usar vendedores dos clientes cadastrados
    const clientes = await crmCarregarClientes();
    const vendedoresUnicos = [...new Set(clientes.map(c => c.vendedor).filter(Boolean))];
    select.innerHTML = '<option value="">Selecione o vendedor</option>';
    vendedoresUnicos.forEach(v => {
      const selected = v === vendedorAtual ? 'selected' : '';
      select.innerHTML += `<option value="${v}" ${selected}>${v}</option>`;
    });
  }
}
```

#### A) Ajustar `formatarData` e `formatarDataInput` para BRT
```javascript
function formatarData(isoString) {
  if (!isoString) return '—';
  const d = new Date(isoString);
  // Ajustar para BRT
  const brt = new Date(d.getTime() - (d.getTimezoneOffset() + 180) * 60000);
  const dia = String(brt.getDate()).padStart(2, '0');
  const mes = String(brt.getMonth() + 1).padStart(2, '0');
  const ano = brt.getFullYear();
  const hora = String(brt.getHours()).padStart(2, '0');
  const min = String(brt.getMinutes()).padStart(2, '0');
  return `${dia}/${mes}/${ano} ${hora}:${min}`;
}

function formatarDataInput(d) {
  const brt = new Date(d.getTime() - (d.getTimezoneOffset() + 180) * 60000);
  const ano = brt.getFullYear();
  const mes = String(brt.getMonth() + 1).padStart(2, '0');
  const dia = String(brt.getDate()).padStart(2, '0');
  const hora = String(brt.getHours()).padStart(2, '0');
  const min = String(brt.getMinutes()).padStart(2, '0');
  return `${ano}-${mes}-${dia}T${hora}:${min}`;
}
```
